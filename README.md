📝 Modern Kanban Task Manager
A professional, responsive Kanban Board application built with TypeScript, HTML5, and CSS3. This project helps users manage their daily tasks efficiently through an intuitive drag-and-drop-like interface with real-time status updates.

🚀 Live Demo
https://kanban-task-manager-mahmoud-hamada.vercel.app

✨ Features
Task Management: Create, edit, and delete tasks with ease.

Dynamic Status Tracking: Categorize tasks into To Do, In Progress, and Completed.

Priority System: Visual indicators for High, Medium, and Low priority tasks.

Due Date Alerts: Special badges for Overdue and Due Soon tasks.

Responsive Design: Fully optimized for Mobile, Tablet, and Desktop using Bootstrap 5.

Local Storage: Your tasks are saved locally in the browser, so they persist even after a page refresh.

Clean UI/UX: Modern interface with smooth transitions and professional color palettes (OKLCH).

🛠️ Tech Stack
Logic: TypeScript (Object-Oriented Programming)

Styling: CSS3, Bootstrap 5, FontAwesome Icons

Structure: HTML5
